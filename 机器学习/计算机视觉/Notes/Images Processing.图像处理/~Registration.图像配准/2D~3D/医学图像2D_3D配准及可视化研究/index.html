医学图像2D_3D配准及可视化研究

ZhuYuanxiang 2023-02-22 09:57:56
Categories： Tags：

医学图像2D_3D配准及可视化研究

[1]刘洋. 医学图像2D/3D配准及可视化研究[D]. 南方医科大学, 2007.

摘要

图像配准：一幅医学图像寻求一种空间变换，从而与另一幅医学图像上的对应点达到空间上的匹配

图像配准是图像融合的前提

Ch01 绪论

在医学成像中，通过连续获取人体某个方向（断层获取方向）上的一系列截面图像来构成一组图像序列，这个图像序列构成了一个包含全部诊断信息的三维离散数据场（即体数据）

Ch02 医学图像配准

2.1 医学图像配准分类

图像特征分类：

  • 外部特征：人造标记
    • 侵入式的：立体定位参考框架、螺钉标记
    • 非侵入式的：牙套、泡沫面具
  • 内部特征：病人的图像信息
    • 基于标记的图像配准
    • 基于分割的图像配准
    • 基于体素特征的图像配准
  • 非图像：通过校准两个扫描器的成像坐标系统进行配准。
    • 前提：两个扫描器在同一个物理位置
    • 要求：获取两幅图像的过程中病人保持不动

基于变换特性分类：

  • 刚体变换：坐标轴的平移和旋转
  • 仿射变换：直线映射成直线，保持了平行性，丢失了垂直性
  • 投影变换：直线映射成直线，丢失了平行性，丢失了垂直性
  • 非线性（曲线）变换：直线映射成曲线

基于变换区域分类：

  • 局部配准：改变配准参数时只影响图像的局部
  • 全局配准：改变配准参数时会影响图像的全局

基于用户的参与程度分类：

  • 全自动式：只提供相应的图像或者信息
  • 半自动式：需执行算法初始化或者指导算法
  • 交互式：用户在软件的帮助下人工执行配准。交互将提高鲁棒性，降低实用性

基于参数求解方式分类：

  • 直接法：把问题简化到一定程度，直接计算得到变换参数。如：穷尽搜索法
  • 基于搜索的方法：从一种或者多种猜测开始，在最优匹配测度的指导下，使得两幅图像在某一变换时达到最大相似

基于图像模态分类：

  • 单模态图像配准
  • 多模态图像配准
  • 模态和（解剖或者生理）模型间的配准
  • 模态和患者之间的配准

基于图像来源分类：

  • 内部：同一患者的图像配准
  • 外部：不同患者的图像配准
  • 患者和图谱间的配准

2.2 图像配准的基本步骤与方法

配准的基本步骤：

  1. 图像特征提取：待配准的两幅图像一般包含能够反映其特性的特征量，如：图像的边界、应解剖的特征点、外标记点等
  2. 特征配对：提取图像特征量以后，需要定义一个目标函数，用来表示两幅图像的相似性，即相似性测度
  3. 选取几何变换，确定参数：根据图像的特征选择合适的空间几何变换方式
  4. 执行变换：变换过程等价于多参数的最优化搜索过程，使用最优化算法找出一组最优变换参数

图像配准的常用方法：

  • 基于点的配准方法
    • 提取图像特征点，对应图像特征点，根据特征点确定空间变换
    • 外部点（Extrinsic）和内部点（Intrinsic）
  • 基于面的配准方法：配准前需要分割，分割精度决定了配准精度
    • 提取图像的曲线或者曲面，根据对应的曲线或曲面决定空间变换
    • 头帽算法 ：从一幅图像中提取一个表面模型称为“头”，从另一幅图像中提取轮廓点集称为“帽”。用刚体变换或者选择性的仿射变换将“帽”的点集变换到“头”上，采用优化算法使“帽”的各点到“头”表面的均方根距离最小
  • 基于矩和主轴的配准方法：对数据缺失较敏感，配准结果不太精确；算法自动、快速、易实现，常用于预配准
    • 计算图像的零阶矩（质心）和一阶矩（主轴），平移和旋转使得两幅图像的质心和主轴对齐完成图像的配准
  • 基于灰度的配准方法：适用于多模态图像配准
    • 根据相似性测度决定代价函数，通过搜索技术使其最小
    • 相似性测度：互信息法、相关性
      • 互信息：两个随机变量统计相关性的测度，两幅图像几何对齐时，它们的对应特征点的互信息最大
      • 相关性：同一物体由于图像获取条件的差异或者物体自身发生的上的改变，采用使图像间相似性最大化的原理实现图像间的配准，即通过优化两幅图像间相似性准则来估计变换参数，所使用的相似性测度：相关函数、相关系数、差值的平方和等

2.3 几何变换

线性变换：

  • 刚性变换：图像中任意两点间的距离在变换前后保持不变
  • 仿射变换：直线映射到直线，平行关系保持不变
    • 各个方向尺度变换系数一致的均匀尺度变换
    • 各个方向尺度变换系数不一致的非均匀尺度变换
    • 剪切变换
  • 投影变换：直线映射到直线，平行关系基本保持不变，常用于三维到二维的投影

非线性变换：直线变为曲线，典型的为多项式变换

2.4 图像插值方法

图像经过几何变换后，新图像的坐标与原始的采样网格坐标不能重合，需要对变换后的图像重新映射到原始的采样网格坐标上，因此需要重采样和插值处理。

常用的插值算法：

  • 最近邻法（nearest neighboring interpolation）：计算量小，速度快；图像质量差
  • 线性插值法（linear interpolation）：计算量适中，图像质量适中
  • 薄板样条插值法（interpolation using thin-plate splines）：用于图像的弹性配准，实现非线性变换和插值

2.5 优化方法

多参数最优化问题：根据具体的配准问题确定一个衡量是否配准或者配准程度的准则，根据配准准则定义一个适当的目标函数，通过对目标的最优化搜索，得到配准参数

最优化过程分类：

  • 参数可以直接计算的最优化：明确的解析表达式，对于特定的配准准则计算的方法和过程都是确定的。只利用了图像中少量的信息
  • 参数需要通过优化搜索的最优化：没有明确的解析表达式，只能通过对目标函数在其定义域上进行优化搜索得到，目标函数定义为一个连续的、光滑的凸函数
    • Powell多维方向集法，本质是共轭方向法
    • 模拟退火算法，又称为Monte Carlo退火法、模拟冷却法、随机松驰法，使用Metropolis算法产生组合优化问题解的序列，以一定的概率接受恶化解，从而有机会从局部极值的陷阱中跳出来，从而收敛到全局最优解
    • 遗传算法，又称为进化算法，找出全局最优的解，适合多参数的优化问题的数值求解

2.6 医学图像配准的评估

由于待配准的多幅图像是在不同时间或者不同条件下获取的，所以没有绝对的配准问题，即不存在金标准（Gold Standard），只有相对的（某种准则下的）最优配准，因此最优配准与目的相关。

常用的评估方法：

  • 体模法：用已知的图像信息验证新配准算法的精度
    • 硬件体模：
    • 软件体模：图像合成结果
  • 立体定向框架系统（Stereotactic Frame Systems）：包括立体定向参考框架、立体定向图像获取、探针或者手术器械导向。优点是定位准确，不易产生图像畸变。因此立体定向框架系统的体积图像数据可以用来评估其它配准方法的精度。
  • 人工记号作准标
  • 领域专家用目标方法检验

4.1 2D/3D医学图像配准基本原理和步骤

目的：术前扫描的三维数据和术中拍摄的二维图像信息中得到术中病人某结构的位置参数，即从二维图像和三维扫描数据信息中找到手术目标的位置

基本步骤：

  1. 根据二维X线影像可以初步估计三维数据在坐标系中的方位
  2. 对三维数据进行处理，生成浮动图像
  3. 采用定义的相似性测度对浮动图像和参考图像进行比较
  4. 在所定义的空间坐标系内对三维数据进行刚性变换

重复2、3步，使得相似性测度达到最大，完成配准。

Ch05 三维可视化系统的研制

5.1 VTK简介

VTK采用流水线（pipeline）机制：

  • Source是流水线的开始，即读取文件产生源数据；
  • Filter（滤波器）对数据作预处理，多源数据输入产生多源数据输出，是一个独立的计算模块
  • Mapper将Filter处理后的数据映射为图形数据，它是可视化流水线和图形模型之间的接口

显示数据时，先建立显示窗口，Actor是显示窗口中的实体，接收Mapper伟过来的数据属性，建立Renderer并用此将结果在最终窗口上显示出来，一个Renderer可以演示多个Actor。

5.2.3 三维重建

三维图像的可视化技术[^29]：

  • 面绘制：体表面的重建，通过平面近似和逼近显示物体表面，基于等值面的面绘制是从切片数据集提供的三维数据中抽取出等值面，再用传统的图形学技术对物体表面进行拟合
  • 体绘制：基于穿过体图像的射线跟踪方法。
    • 基于图像空间顺序的体绘制算法（如：射线追踪算法）
    • 基于对象空间顺序的体绘制算法（如：单元投影法）

等值面：在一个网格空间中由采样值等于某一个给定值的所有点组成的集合

Marching Cube算法是最常用的产生等值面的算法，主要步骤：

  1. 将离散数据场分层读入内存；
  2. 扫描两层数据，逐个构造体元，每个体元中的8个顶点取自相邻两层；
  3. 将体元定点函数值与给定的等值面的值相比较，构造该体元的状态表；
  4. 根据状态表，得出将与等值面有交点的体元边界；
  5. 通过线性插值方法，计算出体元边界与等值面的交点；
  6. 利用中心差分法，求出体元各角点处的法向，再通过线性插值方法，求出三角形各顶点处的法向；
  7. 根据三角面片各顶点的坐标值及法向量绘制等值面图像

VTK提供三种用于射线追踪算法的光强计算函数：

  • 等值面探测函数：重建结果类似于面绘制
  • 最大密度投影函数：与最大密度相关的数据值直接投影到对应屏幕上的每个像素中形成最终结果
  • 合成体绘制函数：最常用的方法，通过分段函数将体数据值转换成颜色、不透明率等光学属性，最后将这些属性合成到屏幕上的像素中形成三维图像
